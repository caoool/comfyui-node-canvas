# Support

Use the issue templates so support requests include enough context to act on.

- Bug report: https://github.com/caoool/comfyui-node-canvas/issues/new?template=bug_report.yml
- Feature request: https://github.com/caoool/comfyui-node-canvas/issues/new?template=feature_request.yml
- Node idea: https://github.com/caoool/comfyui-node-canvas/issues/new?template=node_idea.yml
- Detailed guide: https://github.com/caoool/comfyui-node-canvas/blob/main/docs/wiki/index.md
- Landing page: https://caoool.github.io/comfyui-node-canvas/

For AI Builder issues, include the prompt, provider/model, action log, validation output, and whether deploy permission was enabled.
