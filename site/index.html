Local node-pack IDE

A focused authoring loop for ComfyUI developers.

Custom node work usually means hand-maintaining Python class metadata, generated mappings, frontend display code, dependency files, and deployment steps. This builder puts those pieces into one local workbench while keeping the generated files visible and editable.

Design the node Add typed inputs, outputs, widgets, categories, Return UI, and preview metadata.
Edit the pack Work with Python, requirements, install scripts, custom renderers, and shared files.
Ship locally Export ZIPs or deploy builder-managed packs directly into ComfyUI.

Who it is for

Use it when the node idea matters more than the packaging chores.

The builder is for ComfyUI users who know what behavior they want, but do not want each experiment to start with the same custom-node scaffolding work.

Managed deployment

Built for local ComfyUI development.

Direct deploy writes the active pack to the configured ComfyUI install path, includes builder metadata, installs dependencies when needed, and asks ComfyUI Extension Manager to restart. The hosted landing page is static, but the workbench remains local because those actions need loopback and file-system access. 

<ComfyUI>/custom_nodes/<pack-folder>/
  __init__.py
  builder.project.json
  NodeName.py
  requirements.txt
  web/runtimeUiDisplays.js
